What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, representing the structure and content of an HTML document as a tree of objects. It allows scripts, such as JavaScript, to access and manipulate the elements, attributes, and content of a web page dynamically.

How do you select an element from the DOM?

In JavaScript, you can select an element from the Document Object Model (DOM) using several methods provided by the document object. Each method targets elements based on different criteria such as their id, class, tag name, or other attributes.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of adding separate event listeners to individual child elements. The event listener on the parent element handles events that occur on its child elements through event bubbling, a process where events propagate (or "bubble up") from the target element to the parent elements.
